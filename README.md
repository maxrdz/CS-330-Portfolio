# CS-330 | Comp Graphics & Visualization | Portfolio

### How do I approach designing software?

This project helped me improve my ability to design software by planning out a 3D scene before development and breaking larger objects into smaller components. I learned how to use primitive shapes, textures, lighting, and camera controls to create a complete visual environment. My design process involved analyzing the reference image, identifying the objects needed, creating the models, and refining the scene through testing and adjustments. These design strategies can be applied to future projects by focusing on planning, organization, and incremental improvements.

### How do I approach developing programs?

While developing my 3D scene, I used new strategies such as building objects from reusable components, testing individual elements, and debugging issues through iteration. The development process required making changes, reviewing the results, and improving the program over multiple milestones. My approach to coding evolved by becoming more focused on writing organized, maintainable code and understanding how each part of the program contributed to the final result.

### How can computer science help me in reaching my goals?

Computational graphics and visualization provided me with new skills in creating interactive environments, working with 3D models, and understanding how software can represent real-world objects digitally. These skills can support my future education by expanding my programming knowledge and can help professionally in areas involving software development, simulations, visualization, and interactive applications.
